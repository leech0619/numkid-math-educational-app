# 📱 NumKid

**NumKid** is an interactive and educational mobile application designed to help school children build foundational number skills through fun, game-based learning. Developed using Flutter, the app focuses on four essential mathematics topics with vibrant visuals, real-time feedback, and engaging arcade gameplay.

---

## 🎮 Features

- **Four Core Topics**
  - Comparing Numbers (greater than, less than, equal to)
  - Ordering Numbers (ascending, descending)
  - Composing Numbers (putting numbers together)
  - Counting Numbers (visual-based counting)

- **Arcade Mode**
  - Randomly selected math games
  - Score tracking and high score saving
  - Motivates users to beat their personal best

- **Interactive Feedback**
  - Background music
  - Sound effects for correct and incorrect answers
  - Vibration effect for incorrect responses

- **User-Friendly Interface**
  - Colorful visuals and flip cards
  - Simple navigation tailored for kids
  - Responsive design for various screen sizes

---

## 🛠️ Tech Stack

- **Flutter** – Cross-platform UI toolkit
- **Dart** – Programming language for Flutter
- **SharedPreferences** – Local data storage for high scores
